# Game About Seriously Spinning

## Theme: 
- Twin Stick Shooter
- serious futuristic/Sci Fi (Professional Assassin in a Cyberpunk world?)

## Engine:
Godot (C#/Mono Version) 

## Artstyle: 
2d Pixel/Painted

## Genre:
Rogue-Like

## Gameplay:

### Mandatory Features:
- 2d Character that can move and Rotate
- rotating player by 360° charges rifle
- rifle can fire
- enemies that attack the player
- linear level consisting of 5-10 rooms of escalating difficulty
- Every room fits the screen
- weapon can be upgraded after every room by choosing between 3 random Perks

### Optional Features (from most to least important): 
- final boss at the end 
- procedual level Generation / more Linear Levels
- more enemy types
- more starting weapons
- active items
- unlockable perks/items/weapons
- rooms that dont have to fit the screen
